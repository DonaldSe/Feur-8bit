## 🎮 Contexte du projet

Ce projet a été réalisé dans le cadre de **la Nuit de l’Info 2024**, un événement national durant lequel des équipes d’étudiants développent une application web en une nuit en relevant différents défis proposés par des partenaires.

Le framework choisi est **Angular**, pour sa structure modulaire, sa performance et sa capacité à gérer efficacement une application web moderne.

Nous avons répondu au défi suivant :

### 🎮 Défi “On veut du gros pixel !” – IUT du Littoral Côte d’Opale
**Objectif :**  
Ajouter un élément rétro gaming à une page *ou* transformer entièrement une page pour lui donner un style **8-bit / rétro-gaming**, dans l’esprit des anciennes consoles de jeu.

Ce projet inclut donc une page complète stylisée façon rétro-gaming, respectant l’ambiance des jeux 8-bit.

**Projet réalisé par Feur!**
